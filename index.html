Theme Installation #back to top

There are several ways to install wp themes, we do prefer using FTP but you can choose other ways.
Please read more about theme installation on the official website at http://codex.wordpress.org/Using_Themes.

Upload via FTP:

  • Step 1 - please unzip the installable zip file that you downloaded from ThemeForest (Thenobility.zip).
  • Step 2 - upload "Thenobility" theme folder to the theme directory of your WordPress via ftp.
  • Step 3 - log into your WordPress admin dashboard and activate "Thenobility" theme in the Appearance => Themes section.
  • Step 4 - after succesfull theme activation you will be automatocally redirected to the Thenobility WordPress theme settings.
  • Step 5 - at the top of the page you will see the message 'This theme requires the following plugins: Thenobility Toolkit Plugin and Elementor WordPress Page Builder.   • Step 6 - after succesfull theme and plugins installation you will see the top notification 'Purchase Validation! Please activate your theme', please click the Register now button. It will redirect you to the Product Activation section where you need to enter purchase code.

Theme Update #back to top

Upgrading your website’s theme to the latest version is highly recommended, not only because it ensures you have all the latest features, but you’ll also be up-to-date with any security patches..

Uploading and Replacing an Existing Theme:

Before get started you need to back up your database and your current theme. Since we’re replacing and updating theme files, there’s a chance something could go wrong so it’s best to play it safe and have a backup.

As we uploading and replacing an existing theme don't go to Appearance > Themes > Add New and try to upload a new version of a theme that is already in your site’s themes directory, it will fail and you’ll get an error message telling you the destination folder already exists. The reason for this error is due to the fact you can’t have two folders in your themes directory with identical names (obviously!

Login to your site via FTP or cPanel and delete the old theme and then upload the new version. Make sure you made a backup of the old version of your theme just in case something goes wrong.

After succesfull update please login to your WP Dashboar and check the panel. There could be some notifications about plugins upgrade. The all ones with the theme should be up to date. Please follow the proposed setps to update suggested ones.

Translation#back to top

This theme comes with .pot file for the theme, it is located in \Thenobility\languages\ folder. Go to Loco Translate Please follow these steps to translate your theme text:

  • Step 1 - please download this software and install it http://www.poedit.net/download.php;
  • Step 2 - please open .pot or .po files located in \Thenobility\languages\Thenobility.pot and \locotranslate\languages\Thenobility.pot using poedit tool and translate the existing text to your own;
  • Step 3 - when you save the file, you must name it according to your language;
  • Step 4 - upload the file to the required folder on your server, that's all.

Please note that you can also use the special plugins like WPML to do the translation.
